## 📖 About This Repository
This repository contains my daily solutions to Data Structures and Algorithms (DSA) problems, primarily from LeetCode. The goal is to build a strong foundation in problem-solving by organizing my practice around **Topics** and **Patterns**.

## 🎯 The Goal & The "Reward" System
Consistency is key! To keep myself motivated, I use a reward-based tracking system to gamify my daily problem-solving.

**Reward Tiers based on Daily Streak & Difficulty:**
- 🟢 **Easy Problem** = 🌟 1 Star
- 🟡 **Medium Problem** = 🌟🌟🌟 3 Stars 
- 🔴 **Hard Problem** = 🌟🌟🌟🌟🌟 5 Stars
- 🔥 **7-Day Target** = 🏆 Weekly Champion Badge
- 🚀 **30-Day Target** = 👑 Consistency King/Queen
